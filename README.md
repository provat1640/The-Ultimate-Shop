# SE 322 Project - The Ultimate Shop

This is our group project for SE 322. It's an e-commerce website for buying clothes.

## Features
- Login and registration for users
- Admin panel to add and remove products
- Shopping cart using local storage
- Checkout system
- Dark theme UI

## Technologies used
- HTML, CSS, JavaScript
- Bootstrap 5
- Node.js, Express
- SQLite database

## How to run
1. Clone the project
2. Run `npm install`
3. Run `node server.js`
4. Go to `http://localhost:3000`
